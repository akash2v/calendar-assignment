# Wall Calendar

A wall calendar built with React + TypeScript + Vite. Looks like an actual paper calendar hanging on a wall.

## Features

- Flip animation with sound when switching months
- Click any date to start a range selection, click another to end it
- Add notes per date or date range, stored in localStorage
- Holidays highlighted with a dot indicator
- 5 color themes — Ocean, Forest, Sunset, Rose, Slate
- Click the year or month text in the hero to jump directly via picker
- On mobile, notes panel collapses into a dropdown to keep the calendar visible
- Fully responsive — works on desktop and mobile

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```
